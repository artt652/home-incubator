TotemFTC

Это - первая непубличная альфа версия приложения для учета и управления посещениями фитнес-студии функционального тренинга Totem.

Мобильная версия приложения предназначена записи и для отметки тренировок. Она предназначена для запуска на смартфоне либо из браузера, либо как мобильное приложение. В настоящее время существует только приложение для Android. Если для тестовых целей вы запускаете мобильное приложение на компьютере, то необходимо проделать несложные манипуляции, чтобы эмулировать мобильное приложение и получить возможность использовать тактильные жесты мобильного приложения с помощью обычной мышки - т.н. swipes (свайпы - это когда пальцем водишь по экрану). Для этого в google chrome надо нажать F12, а потом в окне Chrome Tool Window нажать 'Toggle Device Toolbar' или Ctrl + Shift + M и задать размеры мобильного устройства ползунками. Вот чуть более подробные инструкции .

Примечание: Мобильное приложение для iOS тоже можно собрать, но Apple просит за это $100 в год. Без этих денег приложение для iOS будет работать всего неделю.

Админка (Десктопная версия) в теории должна обладать большими возможностями - например создание отчетов, изменение настроек, редактирование расписания и др. действия.

Для того, чтобы начать пользоваться, необходимо запустить одну из версий. Войти. Изменить свое имя в настройках при необходимости (в настоящее время редактирование своего имени работает только в мобильной версии). Затем необходимо написать мне в тележеньку или в чатик, чтобы вам добавили права. После этого можно пользоваться.
