TotemFTC

Это - демо-версия приложения для учета и управления посещениями фитнес-студии функционального тренинга Totem.

Мобильная версия приложения предназначена для посетителей и для отметки тренировок.

Админка - Десктопная версия в теории должна обладать большими возможностями - например создание отчетов, настройки и др. действия.

И, пожалуйста, аккуратнее с личными данными.
