Пользовательское соглашение приложения Totem FTC

TotemFTC — приложение для людей, которые занимаются в фитнес-студии функционального тренинга Totem (г. Химки). В нем можно узнать расписание, записаться на тренировку, связаться с тренером.

Приложение TotemFTC использует персональные данные: ФИО, e-mail, телефон, фото.

Эта персональная информация используется для идентификации пользователей. Также эта информация может быть использована для связи с пользователем.

Приложение TotemFTC не передает персональную информацию о пользователях третьим лицам и не использует ее каким-либо другим способом.
